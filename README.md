# ttysocial
Uma rede social de terminal
atualmente ela suporta:

1: um sistema de correio eletronico chamado T-MAIL (Terminal Mail)

2: Salas de chat públicas

3: Salas de chat privadas com suporte a proteção por senha

4: chat semi-direto(basicamente ele reune todas as pessoas que querem falar com um usuario em uma sala)

5: um Fórum

lembrando que ainda está em progresso, qualuqer bug pode levar a um crash, e o server também não estará ativo todo momento, pois meu PC não aguentaria,qualquer bug basta comentar nesse post (ah, e o codigo sera distribuido daqui a pouco tempo no github)
